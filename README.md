<h1 style="display: flex; justify-content: space-between;">Apex </h1>

Apex is a personalized background program designed to execute specific functions and provide text-based responses using a local AI system.

## Features

- 🎤 **Voice Recognition**  
  Utilizes Vosk speech recognition to continuously monitor audio input and detect keywords.

- ⚡ **Keyword-Based Actions**  
  Matches recognized speech against predefined keywords and triggers associated functions.

- 🧠 **Local AI Integration**  
  Generates text responses using a locally hosted AI model.

- 🔇 **Background Operation**  
  Runs silently in the background until a keyword interrupt is detected.

- 📡 **Bluetooth Interaction**  
  Uses GrooveAPI to communicate with nearby Bluetooth devices.

## How It Works

1. The program continuously listens to an audio input stream.
2. Vosk processes the audio and converts speech to text.
3. The recognized text is scanned for predefined keywords.
4. When a keyword is detected, the corresponding function is executed.
5. Optional AI responses are generated locally.
